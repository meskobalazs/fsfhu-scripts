GNOME-related scripts
=====================

* d-l-modulechecker.sh : Check if the modules listed on https://l10n.gnome.org/module/ are still present in the GNOME git
* gmirr.sh: Download all translatable po files from https://l10n.gnome.org/languages/$YOURLANGUAGE for offline work
* liedetector.sh: Detect differences between the GNOME core release set and the localizable modules listed on https://l10n.gnome.org/releases/gnome-$STABLE
* ui-tr-chk.sh: Check the translatable flag in strings of .ui files
